Behind the scenes

All the projects have been created using HTML5, CSS3 and JavaScript. The CSS has been structured with BEM principle and pre-processed with SASS. All the dependencies have been installed using node package manager(npm) and all files have been combined and injected into appropriate project folders using a task runner called gulp. Few of the jquery plugins and a google map API have been utilised to build steelbeauty.london website. The dom manipulation event has been defined to modify and copy the code to paste in the stylesheet for the gradient page. In the font combination page, the dom event is defined to change the text content to try the fonts displayed.
Currently I am diving deep into advanced concepts of Javascript, and exploring the framework called react. At the same time, I have been keeping myself abreast with the ever improving developers' world using various mediums such as Medium, Youtube, podcasts and email subscriptions.
